## Development

Install all dependencies for a project

```
$ yarn install
```

To mark the source code with some suspicious and non-structural (may cause bug)

```
$ yarn lint
$ yarn lint-fix
```

Run tests

```
$ yarn test
```

Development server at <a href="http://localhost:3000/" target="_blank">http://localhost:3000/</a>

```
$ yarn start
```
